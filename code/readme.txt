To run program you have to:
1) Add PacketTransmission.exe to C:\emu8086\DEVICES folder
2) Run the emu8086 emultor as Administrator (to have access to c:\emu8086.io file)
3) (Optional) If an error occurred about .NET Desktop runtime you must download and install and try step 2 again: https://dotnet.microsoft.com/en-us/download/dotnet/thank-you/runtime-desktop-7.0.5-windows-x64-installer
