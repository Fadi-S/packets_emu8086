1) Add PacketTransmition.exe to C:\emu8086\DEVICES folder
2) If error about .NET runtime show you have to download it from: https://download.visualstudio.microsoft.com/download/pr/dffb1939-cef1-4db3-a579-5475a3061cdd/578b208733c914c7b7357f6baa4ecfd6/windowsdesktop-runtime-7.0.5-win-x64.exe
3) Run the emu8086 emultor as Administrator (to have access to c:\emu8086.io file)
